# AI-TIL

AI, LLM, ML, DL 관련 논문과 코드를 분석하고 학습한 내용을 기록하는 저장소입니다.

## 📚 About

이 레포지토리는 인공지능 분야의 최신 논문과 기술을 깊이 있게 공부하고, 그 내용을 체계적으로 정리하기 위해 만들어졌습니다. 
각 주제는 GitHub Issues를 통해 관리되며, 수학적 증명과 개념을 상세하게 다룹니다.

## 🎯 목표

- 최신 AI/ML/DL 논문의 핵심 개념 이해
- 수학적 기초와 증명 과정 상세 분석
- 이론과 실제 구현의 연결
- 지속적인 학습과 기록
